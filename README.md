# ATT

ATT (Agent Toolkit Toolkit) is a web-based control plane for building and operating NeMo Agent Toolkit apps.

## Quickstart

1. Use Python 3.12 or 3.13 (`nvidia-nat` is not available on 3.14+)
2. Install dependencies with `uv sync --extra dev`
3. Run API server: `uv run att-api`
4. Run tests: `uv run pytest`
